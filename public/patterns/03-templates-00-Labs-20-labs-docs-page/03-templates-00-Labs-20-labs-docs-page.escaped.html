The API root URL is located at:</p> <pre><code>http://www.europeana.eu/api/v2</code></pre> <p>For obligatory request parameters look into the documentation of specific calls. The <a href="http://labs.europeana.eu/api/authentication/">authentication</a> section provides information on the obligatory authentication parameter <em>wskey</em>.</p> </section> <section id="response"> <h2>Response</h2> <p>A response to an API call will always contain a number of standard fields that precede the fields specific for the call. The standard part contains the following fields:</p> <table class="table-responsive"><thead><tr><th>Field</th><th>Datatype</th><th>Description</th></tr></thead><tbody><tr><td>apikey</td><td>String</td><td>the authentication parameter sent out by the client (the wskey parameter)</td></tr> <tr><td>action</td><td>String</td><td>the name of the API method that was called</td></tr> <tr><td>success</td><td>Boolean</td><td>a boolean (true/false) flag denoting the successful execution of the call</td></tr> <tr><td>statsDuration</td><td>Number</td><td>the time (in milliseconds) taken to serve the request</td></tr> <tr><td>requestNumber</td><td>Number</td><td>a positive number denoting the number of request by this API key within the last 24 hours</td></tr> <tr><td>error</td><td>String</td><td>if the call was not successful this fields will contain a detailed text message. See <a href="labs.europeana.eu/documentation/error-codes.html">Error Codes</a> for more information.</td></tr> <tr><td>params</td><td>Object</td><td>original request parameters. If an invalid request parameter was submitted, this response parameter will contain the default value (see individual calls for the default values).
